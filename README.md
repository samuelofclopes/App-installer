App Manager
===========================================================================
O "App Manager" é o gestor de programas multiplataforma.

Ele conta com a possibilidade de instalar, atualizar e desintalar qualquer programa que ele tenha registros.

Funcionalidades
============================================================================
Interface organizada, por ordem alfabética de uma lista de aplicações disponiveis no sistema operativo em que foi executado

[ Windows / Linux / Darwin (MacOS) ]

Podendo desinstalar, instalar e atualizar a aplicação escolhida pelo utilizador.

E ele é extencivel, basta adicionar um código e uma app no sistema certo, e se for rodar, já aparece a aplicação lá direita.

O que ajuda bastante na organização, e, caso quem estiver a ler isto queira dar um pull request, sinta-se á vontade, pois isso vai ajudar bastante!

Como funciona?
============================================================================

A partir de comandos já conhecidos como:

    Windows: Winget
    Linux (UBUNTO): Sudo apt
    MacOS: Brew

Eu ajustei a forma de como esses comandos são executados para uma interface amigável para o utilizador.

Porque eu o fiz?
=============================================================================

Este projeto teve dois objetivos principais para a sua criação.

1 - 

    Facilidade de instalação de aplicações, seja após formatação ou apenas para atualizar e desisntalar quando deixa de ser necessario.
    Sem ter de passar pelo processo chato de "Next, Next, Next, Install, Finish, Close" e ainda ficar com o pc sujo de instaladores.
2 - 

    Eu quero aprender a mexer cada vez mais com o terminal, não só no Linux, mas tambem no Windows.
    E este projeto ajudou-me a entender como funcionavam os packages das aplicações.

ATENÇÃO!!!

Caso vejam que está errado, porfavor avisem, ou mandem um pull request com o certo.
